变量

变量名=值

  1. 变量名不能以数字开头
  2. 等号两边不能有空格
  3. 值用双引号引起来
  4. 定义变量时不加$，引用时需要加$
  5. 在终端敲该命令，所得的变量是当前shell的变量
  6. 单引号引用的变量不会被解析
  7. 这样定义是本地变量，不会影响到子shell
